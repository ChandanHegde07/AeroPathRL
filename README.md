# AeroPath RL

## Project Idea

AeroPath RL is an autonomous drone navigation project using reinforcement learning.

The main idea is to train a drone agent to:
- start from a spawn point,
- understand its surroundings from state and distance-sensor inputs,
- and learn a safe, efficient path to a target location while avoiding collisions.

This project uses PPO-based learning in a 3D simulation setup (AirSim), with a mock fallback environment for development and testing.
