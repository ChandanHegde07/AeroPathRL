Drone RL Training and Evaluation Hub

AeroPath RL trains a drone policy with PPO to navigate toward a goal while avoiding unsafe behavior. Use the dashboard for model-performance analytics and the 3D player for episode-level trajectory replay.

Algorithm PPO with MLP policy for continuous action control.
Environment Local Gym-based drone environment with sensor-driven observations.
Outputs Models, training logs, evaluation stats, and trajectory datasets.
Visualization 2D analytics dashboard + interactive 3D trajectory playback.

Core Capabilities

Policy Learning PPO learns continuous velocity outputs for goal-directed navigation.
Safety Objectives Reward shaping encourages progress while penalizing collision and boundary violations.
Evaluation Tooling Batch evaluation exports trajectories and summary statistics for post-analysis.

System Flow

main.py (CLI) -> train | evaluate | info -> DroneNavigationEnv (state + sensors) -> PPO Policy (predict action vx, vy, vz) -> RewardFunction + episode metrics -> outputs: models/, logs/, eval_results/ -> visualized in dashboard.html and simulation3d.html

Quick Commands

Train
python3 main.py train --timesteps 200000
Evaluate
python3 main.py evaluate --model models/saved_models/final_drone_ppo.zip --mode batch --n 100 --save --out_dir eval_results/
